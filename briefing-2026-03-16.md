# Daily Briefing — March 16, 2026

## Brain Link v2.1 — Major Feature Session

### New Features
- **Grok Imagine Integration**: generate_image (Aurora, ~2-5s) and generate_video (async, 1-3 min) tools added. Both render in preview modal. Images at $0.02 standard / $0.07 pro, video at $0.05/sec.
- **Cinematic Prompt Engine**: Brain Link now writes structured 10-block prompts for Grok — identity preservation, lens specs (focal length in mm), framing, performance directives, negative prompts. Trained on Moe's UGC selfie templates.
- **Image-to-Video Pipeline**: Phone photos upload to Supabase Storage (public URL), auto-passed to Grok video API as image_url. Also works with Grok-generated images. Full flow: pick photo -> resize 1200px -> Supabase upload -> generate video from it.
- **Video/Image Export Fixed**: Export button now downloads videos/images via expo-file-system and opens Android share sheet. Previously only worked for HTML artifacts (PDF export).
- **Gateway Cloud Deploy**: Gateway moved from desktop Cloudflare tunnel (brainlink.mrqwen.us) to always-on Docker container on DigitalOcean (gateway.lock28.com). Auto-restarts on crash/reboot. Built locally, pushed to GHCR, deployed via SSH with Traefik routing.

### Bugs Fixed
- APK signature mismatch (EAS vs local Gradle keystore switch)
- Image upload stuck on "Uploading..." (gateway upload was blocking, now Supabase is primary)
- Images not showing in chat (data: URIs too large for RN Image — switched to Supabase HTTPS URLs)
- Input bar behind Android nav buttons (paddingBottom increased to 56px)
- Keyboard hiding input field (wrapped entire screen in KeyboardAvoidingView)
- Coolify build OOM (bypassed — build locally, push pre-built image)
- Dockerfile npx tsc picking wrong package (use ./node_modules/.bin/tsc)

### Architecture Changes
- New env var: EXPO_PUBLIC_GROK_API_KEY
- New Supabase Storage bucket: brainlink-uploads (public)
- New dependency: expo-image-manipulator (resize before upload), expo-file-system (video download)
- Gateway URL changed: brainlink.mrqwen.us -> gateway.lock28.com
- Gateway Docker image: ghcr.io/mrmoe28/brainlink-gateway:latest
